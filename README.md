# AnnaSlipper
AnnaSlipper is a free python based slipper for roblox allowing you to create unique copies of images with resizing capabilities and you get to choose how many pixels to change
# Features:
* # image preview:
* as you expect it just shows your image
* # Modification Settings
* pixels to change: 1% - 100%
* Number of copies: ammount of copies to make will be saved in a new folder in downloads named AnnaSlipper
** Output size (optional)
* when enabled you can change the size of your output image
